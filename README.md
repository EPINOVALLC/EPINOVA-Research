# EPINOVA Publication Manual v1.0

Global AI Governance and Policy Research Center  
EPINOVA LLC

---

## 1. Overview

EPINOVA adopts a standardized publication system to ensure consistency, traceability, and interoperability across research outputs.

The system defines:
- Publication types
- Thematic series
- Coding structure
- Citation format

---

## 2. Publication Types

| Type | Code | Description |
|------|------|-------------|
| Institutional White Book | IWB | Foundational frameworks and core methodologies |
| Policy Brief | PB | Rapid-response analysis and policy-oriented outputs |
| Working Paper | WP | In-depth analytical and theoretical research |
| Policy Report | PR | Applied policy analysis with structured recommendations |
| Research Report | RR | Empirical or data-driven research outputs |

---

## 3. Working Paper Series

Working Papers are organized into thematic series:

| Series | Code | Scope |
|--------|------|------|
| Conflict & Strategic Dynamics | WP–F | Warfare, escalation, deterrence, strategic systems |
| Digital Infrastructure & Data Governance | WP–D | Data centers, infrastructure, compute geography |
| AI Governance & System Design | WP–A | AI systems, governance models, algorithmic structures |

---

## 4. Coding Structure

All publications follow a standardized identifier: 

EPINOVA–[TYPE]–[SERIES]–[YEAR]–[NUMBER]

Examples:

- EPINOVA–WP–F–2026–09  
- EPINOVA–PB–2026–15  
- EPINOVA–IWB–2026–01  

Notes:
- SERIES is used only for Working Papers
- NUMBER resets annually within each type

---

## 5. Citation Format

### Standard Citation (APA-style adapted)
Wu, Shaoyuan. (YEAR). Title.
(EPINOVA–TYPE–SERIES–YEAR–NUMBER).
Global AI Governance and Policy Research Center, EPINOVA LLC. DOI


Example:


Wu, Shaoyuan. (2026). A Systemic Theory of Escalation and the Loss-of-Control Threshold in Networked Conflict.
(EPINOVA–WP–F–2026–09).
Global AI Governance and Policy Research Center, EPINOVA LLC.
https://doi.org/10.5281/zenodo.19139977


---

## 6. Author Name Standardization

All publications must use a consistent author format:


Wu, Shaoyuan


Notes:
- Do NOT use variations (e.g., Wu Shao-Yuan, S. Wu)
- Institutional publications may use: EPINOVA

---

## 7. Versioning

Publications may include version numbers:

- v0.1 — draft / initial release  
- v1.0 — stable version  
- v2.0+ — major revision  

Example:


AI-Strategic Node Framework (v0.1)


---

## 8. DOI and Archiving

All publications are assigned a DOI via Zenodo.

- DOI is the authoritative reference
- GitHub serves as the structural repository
- Website serves as the presentation layer

---

## 9. Platform Integration

EPINOVA research outputs follow a three-layer structure:

| Layer | Platform | Function |
|------|--------|----------|
| Structure | GitHub | Repository and version control |
| Archive | Zenodo | DOI and long-term preservation |
| Interface | epinova.org | Public-facing research portal |
